## Overview

For organizations that want the benefits of a managed solution with the security of running within their own Azure subscription, our Azure Managed Application option provides an ideal middle ground. This deployment model offers:

- Simplified installation through Azure Marketplace
- Pre-configured architecture following Azure best practices
- Automated updates and maintenance
- Complete data residency within your Azure subscription
- Reduced operational overhead compared to self-hosted options

## Getting Started

### Step 1: Find Impact AI in the Azure Marketplace

1. Log in to the [Azure Portal](https://portal.azure.com)
2. Search for "Impact AI" in the Marketplace
3. Select the Impact AI Platform Managed Application
4. Click "Create" to start the deployment process

### Step 2: Configure the Application

Complete the configuration form with information specific to your organization:

- **Basics**:
  - Subscription: Select the Azure subscription where you want to deploy
  - Resource group: Create a new or select an existing one
  - Region: Choose the Azure region for deployment
  - Application name: Provide a name for your Impact AI instance

- **Authentication**:
  - Authentication type: Microsoft Entra ID (recommended) or Local accounts
  - Admin email: Email address for the administrator account

- **Networking**:
  - Virtual Network: Create new or use existing
  - Subnet: Create new or use existing
  - Access options: Select public, private, or hybrid access

### Step 3: Review and Create

1. Review your configuration settings
2. Accept the terms of service
3. Click "Create" to deploy the managed application

Deployment typically takes 20-30 minutes to complete.

## Post-Deployment Configuration

After the application is deployed:

1. Navigate to the deployed managed application in your Azure Portal
2. Access the application URL provided in the Outputs tab
3. Log in with the administrator credentials you configured
4. Complete the initial setup wizard
5. Start connecting your AI applications using the [Connect guide](/quickstart)

## Resource Governance

The Azure Managed Application deploys the following resources in your subscription:

- Container Apps for application services
- PostgreSQL Flexible Server for database
- Key Vault for secure secrets management
- Storage Account for data storage
- App Configuration for settings management
- Virtual Network for network isolation

These resources are deployed in a dedicated resource group that is managed by Impact AI, while you maintain visibility and monitoring capabilities.
