## Get Started

**Expected time:** 1 minute

1. Create an account on [impact.ai](https://impact.ai).
2. Login to the [Impact AI](https://impact.ai) platform.
3. [Laila](/workflows/laila) will guide you through adding your first product.

Congratulations, your account is now set up!


## Next Steps

- **Get started:** Add a product with [Laila](/workflows/laila).
- **Add Traces:** Start adding Traces using the [Connect](/quickstart) guide.
