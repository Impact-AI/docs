## Data Analysis

Orion helps you understand your data:

### Asking Questions

```
You: Why did our quality score drop last week?

Orion: I'll analyze the quality score drop.

**Analysis Period:** Last 7 days vs previous 7 days

**Finding:**
Quality score dropped from 4.2 to 3.8 (-9.5%)

**Root Cause Analysis:**

1. **Topic Distribution Shift** (Primary factor)
   - Technical questions increased 40%
   - Technical topics score 3.5 vs 4.3 for general

2. **Response Length** (Secondary factor)
   - Average response length increased 25%
   - Longer responses correlate with lower scores

3. **Model Change?**
   - No deployment changes detected
   - Same model version throughout

**Recommendation:**
Focus on improving technical question handling. 
Consider:
- Expanding technical knowledge base
- Adding targeted technical modules
```

### Exploring Data

```
You: Show me the breakdown of user issues this month

Orion: Here's the issue breakdown for this month:

**Topic Distribution:**
| Topic | Count | % | Trend |
|-------|-------|---|-------|
| Product Questions | 2,450 | 35% | ↑ 5% |
| Billing Issues | 1,680 | 24% | ↔ stable |
| Technical Support | 1,400 | 20% | ↑ 12% |

| Other | 490 | 7% | ↔ stable |

**Insights:**
- Technical support growing fastest (12% increase)
- Product questions remain largest category
- Shipping issues declining (seasonal?)

Want me to drill into any category?
```

## Trend Detection

### Identifying Trends

```
You: What trends should I be aware of?

Orion: Here are the key trends I've identified:

**📈 Positive Trends:**

1. **Response Quality** (+8% over 30 days)
   - Helpfulness score improving steadily
   - Fewer "not helpful" flags

2. **Resolution Rate** (+5%)
   - More conversations ending with resolution
   - Fewer escalations to human agents

**📉 Concerning Trends:**

1. **Technical Query Performance** (-15%)
   - Growing topic but declining scores
   - Potential knowledge gap

2. **Response Time** (+200ms)
   - Slight increase in latency
   - May impact user satisfaction

**🔄 Stable Metrics:**
- Safety scores (consistent 4.8/5)
- Topic classification accuracy (92%)

Shall I create alerts for any of these trends?
```

### Anomaly Detection

```
Orion: ⚠️ Anomaly Detected

**What:** Sudden spike in "hallucination" flags
**When:** Yesterday, 2pm-4pm UTC
**Magnitude:** 300% above normal

**Analysis:**
- Affected 45 interactions
- All in "product specs" topic
- Specific product SKU mentioned

**Possible Causes:**
1. New product added without knowledge update
2. Outdated information in knowledge base
3. Model confusion with similar products

**Recommendation:**
Review knowledge base for product SKU #12345. 
Update documentation if needed.
```

## Impact Assessment

### Measuring Changes

```
You: What was the impact of our prompt change last week?

Orion: **Impact Assessment: Prompt Change**
Deployed: Jan 25, 2025

**Before vs After Comparison:**

| Metric | Before | After | Change |
|--------|--------|-------|--------|
| Helpfulness | 3.9 | 4.3 | +10.3% ✓ |
| Clarity | 3.7 | 4.1 | +10.8% ✓ |
| Conciseness | 4.2 | 3.8 | -9.5% ⚠ |
| Response Time | 1.1s | 1.4s | +27% ⚠ |

**Analysis:**
The prompt change improved quality but at the cost of:
- Longer responses (less concise)
- Increased latency

**Statistical Significance:** 
✓ Changes are statistically significant (p < 0.05)

**Recommendation:**
Consider a balanced version that maintains quality 
gains while controlling response length.
```

### A/B Test Analysis

```
You: Analyze our current A/B test

Orion: **A/B Test Analysis**
Test: "New System Prompt" (v2.0-test vs v1.5.0)

**Traffic Split:** 50/50
**Duration:** 5 days
**Sample Size:** 3,200 interactions per variant

**Results:**

| Metric | Control (v1.5) | Test (v2.0) | Δ |
|--------|----------------|-------------|---|
| Quality | 4.1 | 4.4 | +7.3% ✓ |
| Resolution | 76% | 82% | +7.9% ✓ |
| Latency | 1.1s | 1.2s | +9.1% |

**Statistical Analysis:**
- Quality improvement: Significant (p=0.003)
- Resolution improvement: Significant (p=0.01)
- Latency increase: Not significant (p=0.12)

**Recommendation:**
✅ Deploy v2.0 to production
The quality and resolution improvements outweigh 
the minor latency increase.
```
