Impact AI is an evaluation and observability platform for AI products.

The fastest path is:

1. Create your workspace and API key
2. Install the Python SDK
3. Send traces from your app
4. Use Impact Studio and evaluation modules to measure quality

<img
  className="rounded-xl"
  src="/images/intro.png"
  alt="Impact AI Platform"
/>

## Start Here

<Steps>
  <Step title="Sign up and create a workspace">
    Go to [beta.impact.ai](https://beta.impact.ai), complete onboarding, and create your first product/version.
  </Step>
  <Step title="Generate an API key">
    In workspace settings, create an API key for the version you want to instrument.
  </Step>
  <Step title="Install the Python SDK">
    Follow the [quickstart](/quickstart) to initialize `impact` and send your first traces.
  </Step>
  <Step title="Configure evaluation">
    Use [Impact Studio](/platform/impact-studio) and [Impact Eval](/tools/impact-eval) to turn traces into measurable quality signals.
  </Step>
</Steps>

## What You Get

<CardGroup cols={2}>
  <Card title="Observability" icon="route">
    OTLP-based trace and log ingestion from your app with OpenTelemetry-compatible SDKs.
  </Card>
  <Card title="Evaluation" icon="check-double">
    Module-driven scoring and classification across interactions and traces.
  </Card>
  <Card title="Analysis" icon="chart-mixed">
    Metrics, trends, and issue discovery in Analyse and Verify views.
  </Card>
  <Card title="Tooling" icon="wrench">
    Studio tools such as deep research and scoring preview for faster module design.
  </Card>
</CardGroup>

## Product Areas

- **Analyse**: Explore traces, datasets, modules, and issues.
- **Context**: Define product strategy and framework metadata.
- **Verify**: Track module performance for quality, capabilities, and outcomes.
- **Impact Studio**: Build and store modules with assisted tooling.
