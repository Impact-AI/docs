The SDK attempts best-effort auto-instrumentation at `impact.init()`.

If a provider package is not installed, it is skipped without failing your app.

## Model Providers

- OpenAI
- AWS Bedrock
- Anthropic
- Cohere
- Mistral
- Watsonx
- Groq
- Together
- Ollama
- Google GenAI
- Vertex AI (legacy/deprecating path)

## Agent and Orchestration Frameworks

- OpenAI Agents
- Azure AI Agents / Foundry Agents
- Microsoft Agent Framework
- LangChain
- LlamaIndex
- CrewAI
- Agno
- Haystack
- Pydantic AI

## Tooling and Infra Instrumentation

- HTTPX client spans
- Azure AI Search instrumentation
- MCP instrumentation

## Notes on Coverage

- Coverage depends on upstream instrumentation package behavior.
- Some frameworks emit richer logs/events when OTLP logs are enabled.
- In mixed environments, dependency conflicts are handled as non-fatal skips.

## Example

```python
import impact

impact.init()  # registers available provider instrumentations
```

## Disable Specific Instrumentations

Use OpenTelemetry disable flags when needed, for example:

```bash
export OTEL_PYTHON_DISABLED_INSTRUMENTATIONS=openai,anthropic
```
