## Coverage Notes

- Coverage depends on upstream instrumentation package behavior.
- Some frameworks emit richer logs/events when OTLP logs are enabled.
- Dependency conflicts are handled as non-fatal skips.
- Check `instrumentationResults` (JS) or logs (Python) to verify what loaded.

## Example

<CodeGroup>

```python Python
import impact

impact.init()
```

```typescript JavaScript
import impact from "impact-ai";

impact.init();
console.log(impact.instrumentationResults);
```

</CodeGroup>
