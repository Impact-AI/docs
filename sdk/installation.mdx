## Requirements

- Python 3.10 or higher
- pip, uv, or poetry

## Install the SDK

<CodeGroup>

```bash pip
pip install impact
```

```bash uv
uv add impact
```

```bash poetry
poetry add impact
```

</CodeGroup>

## What's Included

The SDK includes instrumentation for all major GenAI providers out of the box:

```
impact
├── opentelemetry-api
├── opentelemetry-sdk
├── opentelemetry-exporter-otlp-proto-http
├── opentelemetry-instrumentation-openai-v2
├── opentelemetry-instrumentation-anthropic
├── opentelemetry-instrumentation-google-genai
├── opentelemetry-instrumentation-vertexai
├── opentelemetry-instrumentation-botocore
├── opentelemetry-instrumentation-ollama
├── opentelemetry-instrumentation-mistralai
├── opentelemetry-instrumentation-cohere
├── opentelemetry-instrumentation-groq
├── opentelemetry-instrumentation-llamaindex
└── ... and more
```

<Note>
  You don't need to install individual instrumentation packages. The Impact SDK includes everything.
</Note>

## Environment Variables

Configure the SDK using environment variables:

| Variable | Required | Description |
|----------|----------|-------------|
| `IMPACT_API_KEY` | Yes | Your Impact AI API key |
| `IMPACT_BASE_URL` | Yes | Your Impact AI endpoint URL |

```bash
export IMPACT_API_KEY="your-api-key"
export IMPACT_BASE_URL="https://your-tenant.impact.ai"
```

## Basic Setup

Add this to your application's entry point:

```python
from impact.sdk import init

# Initialize after importing your AI provider SDKs
init()
```

The SDK will automatically read `IMPACT_API_KEY` and `IMPACT_BASE_URL` from environment variables.

## Explicit Configuration

You can also pass configuration explicitly:

```python
from impact.sdk import init

init(
    api_key="your-api-key",
    api_endpoint="https://your-tenant.impact.ai"
)
```

## Verify Installation

Test that the SDK is working:

```python
from impact.sdk import init
from openai import OpenAI

# Initialize
init(
    api_key="your-api-key",
    api_endpoint="https://your-tenant.impact.ai"
)

# Make a test call
client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "Say hello"}]
)

print(response.choices[0].message.content)
```

Check your Impact AI dashboard — you should see the trace appear within seconds.

## Import Order

<Warning>
  The `init()` call must come **after** importing your AI provider SDKs for auto-instrumentation to work.
</Warning>

```python
# Correct order
from openai import OpenAI          # 1. Import AI providers first
from anthropic import Anthropic
from impact.sdk import init        # 2. Import Impact SDK
init()                             # 3. Initialize

# Now use your AI providers
client = OpenAI()
```
